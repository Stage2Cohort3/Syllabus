Intro to Classes and Objects

C# is an Object-Oriented Programming Language

C# is an Object-Oriented Programming Language

Object-Oriented Programming is a model where programs are organized around objects and data

Software is divided into small units called objects

Data and Functions are built around these objects

Classes

Classes are the core of OOP

A Class is a Blueprint

  • It contains the variables for storing data
  • It contains the functions to perform operations on that data

An Example of a class


using System;

namespace E360Service.DataContracts
{
    public class Borrower
    {
        public string FirstName { get; set; }
        public string MiddleName { get; set; }
        public string LastName { get; set; }

        public string WorkPhone { get; set; }
        public string CellPhone { get; set; }
        public string HomeEmail { get; set; }
        public string WorkEmail { get; set; }
    }
}

Let's Add Some Functions


using System;

namespace E360Service.DataContracts
{
    public class Borrower
    {
        public string FirstName { get; set; }
        public string MiddleName { get; set; }
        public string LastName { get; set; }
        public string HomePhone { get; set; }
        public string WorkPhone { get; set; }
        public string CellPhone { get; set; }
        public string HomeEmail { get; set; }
        public string WorkEmail { get; set; }

        public string GetFullName()
        {
            return FirstName + " " + LastName;
        }

        public string GetAPhoneNumber()
        {
            if(CellPhone != "")
            {
                return CellPhone;
            }
            else if(HomePhone != "")
            {
                return HomePhone;
            }
            else {
                return WorkPhone;
            }
        }
    }
}

Classes are NOT objects!

Classes are just a blueprint for making objects.

To make an object, just new it up!

Making a New Object


using System;

namespace MyApplication
{
    public class Program
    {
        public void Main()
        {
            var myBorrower = new Borrower();
        }
    }
}
